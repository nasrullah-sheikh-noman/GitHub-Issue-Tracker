## Question 1: What is the difference between var, let, and const?

# Ans: Var is a global-scoped variable, it can be redeclared and reassigned. Let is block-scoped, it can't be redeclared but it can be reassigned. Const is also block-scoped, it can't be redeclared or reassigned.



## Question 2: What is the spread operator (...)?

# Ans: Spread operator (...) is used to expand elements of an array or properties of an object.



## Question 3: What is the difference between map(), filter(), and forEach()?

# Ans: map() returns a new array after tranforming each element. filter() returns a new array with elements that match a condition. forEach() does not return a new array, it's only loops in the array.



## Question 4: What is an arrow function?

# Ans: An arrow function is an shorter way to write a function using arrow(=>) symbol.



## Question 5: What are template literals?

# Ans: Template literals is a way to create a strings using backticks(``) that allow embedded variables and expressions using ${}. They also support milti-line strings.
